# Tesla & GameStop Stock Dashboard

Proyecto de análisis financiero usando Python.

## Objetivos
- Extraer datos históricos de acciones
- Obtener datos de ingresos mediante web scraping
- Visualizar tendencias con gráficos

## Tecnologías utilizadas
- Python
- yfinance
- BeautifulSoup
- Pandas
- Matplotlib

## Acciones analizadas
- Tesla (TSLA)
- GameStop (GME)

Este proyecto forma parte de un laboratorio de análisis de datos financieros.
